##任务目标
- 练习综合运用HTML、CSS、JavaScript实现局部功能
- 练习对于代码的抽象与封装
- 为第四阶段的RIA任务做准备

##任务描述
- 基于任务40，进行如下图中的升级 
- ![设计图](http://7xrp04.com1.z0.glb.clouddn.com/task_3_41_1.jpg)
